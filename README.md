How to run the program

-> Open CMD and move to the cloned directory

-> Run the program with

      pyhton tunts_challenge.py
 
-> It is required to have Python 3.9.1 and Google Client Library installed to run this script. To install Google Client library:

      pip install --upgrade google-api-python-client google-auth-httplib2 google-auth-oauthlib
      
-> Log in to a Google Account required to run the program
